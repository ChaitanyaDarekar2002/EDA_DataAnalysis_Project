# Personal Lifestyle & Financial Analysis Dashboard

This project analyzes personal lifestyle and financial data using **Python** and **Tableau**. The goal is to discover insights about income, expenses, savings, BMI, and lifestyle habits.

---

## Tools Used

* Python
* Pandas
* Matplotlib
* Tableau

---

## Project Process

* Data Cleaning
* Exploratory Data Analysis (EDA)
* Data Visualization
* KPI Analysis
* Dashboard Creation

---

## Dashboard Insights

* Average Income Analysis
* Expense Distribution
* Savings Rate Analysis
* BMI Category Distribution
* Occupation vs Income
* Lifestyle Habit Analysis
